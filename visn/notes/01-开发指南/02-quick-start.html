# 快速上手  Quick Start

# 安装

使用 npm 构建前，请先确保您已经安装好了 Node.js 运行时环境。

# 通过 npm 安装

npm i visn -S

# 通过 yarn 安装

yarn add visn -S

# 使用

# 引入组件

所有组件文档中的引入路径均以 npm 安装为例，如果你是通过下载源代码的方式使用 Visn，请将路径修改为项目中 Visn 所在的目录。

# 全局注册

在您的Vue项目中，进入main.js文件，进行全局注册：

// 引入组件库和样式
import visn from 'visn/packages'
import 'visn/lib/visn.css'
// 注册组件库
Vue.use(visn)

# 局部注册

同上，局部注册则没有引入整个Visn的必要，只需要局部引入即可，不过需要再声明一下Vue，建议全局注册会方便很多。

import Vue from 'vue'
import VsButton from 'visn/packages/vs-button'
VsButton.install(Vue)
