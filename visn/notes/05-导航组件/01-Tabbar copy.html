# Tabbar 导航栏

# 介绍

为页面提供导航功能，常用于页面底部。

# 引入

在 main.js 中引入，详细介绍见快速上手。

# 使用

引入组件后，可以在Vue文件中使用

<vs-nav></vs-nav>
1

# 基础用法

<vs-nav
  title="标题"
  left-text="返回"
  right-text="按钮"
  left-arrow
  bind:click-left="onClickLeft"
  bind:click-right="onClickRight"
/>
Page({
  onClickLeft() {
    wx.showToast({ title: '点击返回', icon: 'none' });
  },
  onClickRight() {
    wx.showToast({ title: '点击按钮', icon: 'none' });
  },
})

# 按钮插槽

组件使用了Vue提供的插槽语法，支持插入文字，标签，还可以插入组件，默认文字为A Button。

<vs-nav></vs-nav>	<!--默认文字A Button-->
<vs-nav>文字</vs-nav>	<!--文字-->
<vs-nav><span>span标签</span></vs-nav>	<!--插入span标签-->
<vs-nav><vs-loading></vs-loading></vs-nav>	<!--搭配visn的loading组件，表现加载中的效果-->

# 按钮尺寸

字符串类型，支持自己定制尺寸大小，有长length和宽width两个属性，默认为16*8（单位：px），通过绑定属性传入。

<vs-nav>默认按钮</vs-nav>
<vs-nav length="40px">长40px按钮</vs-nav>
<vs-nav width="24px">宽24px按钮</vs-nav>
<vs-nav length="40px" width="24px">40*24(px)按钮</vs-nav>

# 自定义颜色

允许直接输入颜色的英文，或者输入对应RGB值（前加井号#）或RGBA值，默认为blue。

<vs-button :color="'red'">红色按钮</vs-button>
<vs-button :color="'#abc'">RGB</vs-button>
<vs-button :color="'#abcd'">RGBA</vs-button>

# 按钮边框

布尔值类型，可以调整边框的有无，边框的默认样式为黑色2px粗，不支持定制 。

<vs-button>自带边框</vs-button>
<vs-button :border="true">边框开启</vs-button>
<vs-button :border="false">无边框</vs-button>
