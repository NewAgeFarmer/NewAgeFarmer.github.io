# Sidebar 侧边栏

# 介绍

为页面提供导航功能，用于页面的侧边。

# 引入

在 main.js 中引入，详细介绍见快速上手。

# 使用

引入组件后，可以在Vue文件中使用

<vs-sidebar></vs-sidebar>
1

# 代码演示

# 基础用法

传入一个数组作为侧边栏的选项，数组里每一个成员为对象。

<vs-sidebar :width="'50px'" ></vs-sidebar>
<vs-sidebar :itemHeight="'12px'" :width="24px"></vs-sidebar>
1
2

# 定制尺寸

定制侧边栏的宽度和里面每一个选项的高度。

<vs-sidebar :width="'50px'" ></vs-sidebar>
<vs-sidebar :itemHeight="'12px'" :width="24px"></vs-sidebar>
1
2

# 自定义颜色

自定义选中与未被选中的颜色。

<vs-sidebar :color="'red'">红色按钮</vs-sidebar>
<vs-sidebar :color="'#abc'">RGB</vs-sidebar>
<vs-sidebar :color="'#abcd'">RGBA</vs-sidebar>
