# Icon 图标

# 介绍

图标组件，使用了阿里巴巴旗下的iconfont图标库，基于字体的图标集，可以通过Icon组件使用。

# 引入

在 main.js 中引入，详细介绍见快速上手。

# 使用

引入组件后，可以在Vue文件中使用

<vs-icon></vs-icon>
1

# 代码演示

# 按钮插槽

组件使用了Vue提供的插槽语法，支持插入文字，标签，还可以插入组件，默认文字为A Button。

<vs-icon></vs-icon>	<!-- 默认为Home标签 -->
<vs-icon>标签右边显示文字</vs-icon>	<!--通过插槽可以在图标右边插入文字-->
<vs-icon icon="上传">上传</vs-icon>	<!--通过icon属性改变图标，具体看下方表格-->
1
2
3

# 图标尺寸

字符串类型，icon被设计为正方形，长宽必须相等，仅支持通过size属性调整边长，需带单位px。

<vs-icon>默认24px*24px</vs-icon>
<vs-icon size="36px">边长36px*36px</vs-icon>
1
2

# 自定义颜色

支持修改图标的颜色（但不改变背景颜色，如需修改背景颜色需要在外层套一个盒子）。

<vs-icon color="red">红色图标</vs-icon>
<vs-icon color="#abc">RGB</vs-icon>
<vs-icon color="#abc4">RGBA</vs-icon>
